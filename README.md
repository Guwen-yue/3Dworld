## ✨ 项目简介
> 本文《当业务一复杂的时候，Prompt 就无从下手？用一个 3D 世界编辑器带你玩转 Prompt》的配套示例工程，用于演示**复杂业务结构化 Prompt 设计、AI Coding 落地、Three.js 3D 场景编辑器开发**的完整实现流程，包含可直接双击运行的纯前端源码，零配置、零构建、开箱即用。

## 🎯 功能/效果预览
基于纯前端 + Three.js 实现的桌面级 3D 世界编辑器，支持可视化编辑、数据持久化、随机村庄生成，全程由结构化 Prompt 驱动开发完成。
## 🚀 快速上手
### 1. 克隆仓库
```bash
git clone https://github.com/Guwen-yue/3Dworld.git
cd 3Dworld
```

### 2. 直接运行
无需安装依赖、无需编译构建，**直接双击打开项目中的 `index.html` 文件**，即可在浏览器中运行 3D 编辑器。

### 3. 核心功能
- 8x8 网格 3D 世界可视化编辑
- 7 种元素一键放置：草地、土路、水、石头、树、房子、擦除
- 鼠标拖拽旋转视角、滚轮缩放、格子悬停反馈
- 3 个存档自由切换，本地持久化保存数据
- 随机生成程序化小村庄 / 一键清空世界
- 新手操作提示自动淡出
- 可以进

## 📁 项目结构
```
3Dworld/
├── index.html       # 主页面结构
├── style.css        # 页面样式与 UI 风格
└── main.js          # 3D 场景、交互、存档、生成逻辑核心代码
```

## 💡 使用说明
- 鼠标左键：点击格子放置/删除元素
- 鼠标拖拽：旋转 3D 视角
- 鼠标滚轮：缩放场景
- 存档按钮：切换不同世界数据
- 重置按钮：随机生成新的村庄地图
- 清空按钮：一键重置为全草地世界
